## 快手视频解析

---

基于`Node.js`构建的爬虫，能够帮助你一键下载你喜欢的主播主页视频，主要用到的插件和依赖包有：

```js
    "cookie-parser": "~1.4.3",
    "debug": "~2.6.9",
    "express": "~4.16.0",
    "http-errors": "~1.6.2",
    "jade": "~1.11.0",
    "morgan": "~1.9.0",
    "request": "^2.85.0",
    "request-promise": "^4.2.2"
```
